1.시민의 수는 고정하고 마피아 수를 늘리면 시민이 이길 확률은 거의 항상 줄어든다
2.마피아의 수를 고정하고 시민의 수를 늘리면 항상 시민이 이길 확률이 올라갈 것 같지만 사실 그렇지 않다
-> 시민의 수가 홀수일때와 짝수일때 따로따로 봐야한다
-> 홀수일떄를 따로 보면 시민의 수가 3, 5, 7, 9로 증가할때마다 시민이 이길 확률이 올라간다
-> 짝수일때도 마찬가지로 시민의 수가 2,4,6,8로 증가할때마다 시민이 이길 확률이 올라간다
하지만 홀수 짝수는 완전히 독립적인 사건으로 봐야한다
즉 예로 마피아 수가 1명으로 동일하고 
시민의 수가 5명에서 6명으로 증가할때 시민이 이길 확률은 오히려 감소하는 것을 볼 수 있다

게임의 구조 : 낮에 시민+마피아 중 한명이 죽고 밤에 시민 중 한명이 죽는다
무조건 시민의 명수가 커진다고 시민이 이길확률이 커진다고 볼 수 없는게
원래 5명이었다가 6명이 되면
첫 낮에 시민이 죽을 확률 : 5/6에서 6/7이 된다. 오히려 시민이 죽을 확률이 올라감을 알 수 있다
둘 낮에 시민이 죽을 확률 : 5/6 * 4/5  == (2/3) 에서 6/7 * 5/6 == (5/7) 여전히 시민이 죽을 확률이 더 높다
셋 낮에 시민이 죽을 확률 : 5/6 * 4/5 * 3/4 == (1/2) 에서 6/7 * 5/6* 4/5 == (4/7) 여전히 시민이 죽을 확률이 더 높다
넷 낮에 시민이 죽을 확률 : 5/6 * 4/5 * 3/4 * 2/3 == (1/3) 에서 6/7*5/6*4/5*3/4 == (3/7) 여전히 시민이 죽을 확률이 더 높다

원래 6명이었닥 7명이 되면
첫 낮에 시민이 죽을 확률 : 6/7에서 7/8이 된다
둘 낮에 시민이 죽을 확률 : 6/7 * 5/6 == (5/7) 에서 7/8 * 6/7 ==(3/4) 여전히 시민이 죽을 확률이 더 높다
